# IoT-Enabled-Energy-Monitoring-System
I built a smart energy monitoring system that tracks electricity usage in real time and displays it on a mobile app using Blynk. This helps users monitor and manage their energy consumption remotely.

The system provided real-time monitoring of voltage, current, and power through the Blynk app. The output showed stable voltage levels and varying power based on load, with reliable data transmission and effective visualization for energy analysis.
